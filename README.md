# State Interaction Spin-Orbit (SISO) Method for SF-GNOCI 
Implementation of state interaction spin-orbit (SISO) coupling method using configuration interaction (CI) vectors from grouped bath ansatz for spin-flip non-orthogonal configuration interaction (SF-GNOCI)

# Installation
*Prerequisites
  *[PySCF](https://github.com/pyscf/pyscf) : 2.10 or higher
  *[PySCF-forge](https://github.com/pyscf/pyscf-forge) : #126 or higher
  *[fci-siso](https://github.com/hczhai/fci-siso)
*Add sfnocisiso top-level directory to your PYTHONPATH e.g. if sfnocisiso_preview is installed in /opt, your PYTHONPATH should be
'''
export PYTHONPATH=/opt/sfnocisiso_preview:$PYTHONPATH
'''
