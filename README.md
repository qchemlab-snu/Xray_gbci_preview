# Xray_gnoci 
A library of ab-initio X-ray spectrum simulation using CAS-GNOCI

# Installation
* Prerequisites
  * [PySCF](https://github.com/pyscf/pyscf) : 2.10 or higher
  * [pyscf-forge](https://github.com/pyscf/pyscf-forge) : #126 or higher
  * [fci-siso](https://github.com/hczhai/fci-siso)
* Add sfnocisiso top-level directory to your PYTHONPATH e.g. if Xray_gnoci_preview is installed in /opt, your PYTHONPATH should be
```
export PYTHONPATH=/opt/Xray_gnoci_preview:$PYTHONPATH
```
