# Xray_gbci 
A library of ab-initio X-ray spectrum simulation using GBCI

# Installation
* Prerequisites
  * [PySCF](https://github.com/pyscf/pyscf) : 2.10 or higher
  * [pyscf-forge](https://github.com/pyscf/pyscf-forge) : #126 or higher
  * [fci-siso](https://github.com/hczhai/fci-siso)
* Add xray_gbci top-level directory to your PYTHONPATH e.g. if Xray_gbci_preview is installed in /opt, your PYTHONPATH should be
```
export PYTHONPATH=/opt/Xray_gbci_preview:$PYTHONPATH
```
